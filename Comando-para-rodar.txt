Compile todos os arquivos:
    javac common/*.java server/*.java client/*.java

Inicie o servidor RMI:
    rmiregistry
        ou
    start rmiregistry 

Execute o servidor:
    java server.ServerMain

Abra 3 terminais separados e execute o cliente em cada um:
   java client.ClienteMain
 


 JogoDaVelhaRMI/
├── src/
│   ├── common/           → Interface remota (contrato entre cliente e servidor)
│   ├── server/           → Implementação do jogo e código do servidor
│   ├── client/           → Interface gráfica dos jogadores


Como Funciona o Ciclo do Jogo:

    O servidor inicia e espera conexões.

    Três clientes (jogadores) se conectam e recebem IDs únicos (1 a 3).

    O primeiro jogador faz uma jogada:

        O servidor valida se é a vez dele.

        Marca a posição e passa a vez para o próximo.

    O próximo cliente pode jogar.

    O jogo continua até que:

        Um jogador alinhe 3 marcas.

        O tabuleiro encha (empate — ainda não tratado no código, pode-se adicionar).

    A vitória é detectada pelo servidor e mostrada aos clientes.
